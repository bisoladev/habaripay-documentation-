# Value Added Services

<p className="text-[#8899a8] font-normal text-xl">These are suites of API that allows you vend airtime and data. Please note that the equivalent value of all data and airtime vended will be charged from your SQUAD WALLET. </p>

### API KEYS (Test Environment)
1. Create an account on our sandbox environment: sandbox.squadco.com
2. Retrieve keys from the bottom of the Get Started Page 

:::caution Authorization
Any request made without the authorization key (secret key) will fail with a 401 (Unauthorized) response code.
:::

:::info
The authorization key is sent via the request header as Bearer Token Authorization
:::

**Example:**
Authorization: Bearer sandbox_sk_94f2b798466408ef4d19e848ee1a4d1a3e93f104046f

### GO LIVE
To go live, simply: 

1. Change the base URL for your endpoints from sandbox-api-d.squadco.com to 
api-d.squadco.com
2. [Sign up on our live enviroment](http://dashboard.squadco.com/)
3. Complete your KYC
4. Use the secret keys provided on the dashboard to authenticate your live transactions
